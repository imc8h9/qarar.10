## 2. الألوان — دلالية، لا زخرفية

**الهوية محفوظة بالكامل.** أُضيفت طبقة دلالية فوقها.

```css
/* الهوية (لم تتغير) */
--ink:#10172A;      --violet:#6D5DFB;   --coral:#FF7A59;
--mint:#27C7A8;     --amber:#F4A261;    --sand:#F7F1E8;

/* دلالية */
--primary:#6D5DFB;         --primary-hover:#5B4BE8;
--success:#11806E;         /* تباين 4.9:1 — نص آمن */
--danger:#C85435;          /* تباين 4.6:1 */
--warning:#8A4B22;
```

**🔴 قاعدة: اللون ليس المعنى الوحيد أبدًا.**
كل حالة مالية = **لون + أيقونة + نص + إشارة**.

| الحالة | اللون | الأيقونة | النص | الإشارة |
|---|---|---|---|---|
| دخل | أخضر | ＋ | «دخل» | + |
| مصروف | مرجاني | − | «مصروف» | − |
| آمن | أخضر | ✓ | «فوق الحد الآمن» | — |
| خطر | مرجاني | ⚠️ | «تحت الحد الآمن» | — |

## 3. الخط

**`IBM Plex Sans Arabic` — أُبقي عليه.**
**السبب:** خط عربي حديث، أوزانه كاملة، مقروء في الأحجام الصغيرة، ويحمل طابعًا
مؤسسيًا يناسب المال. لا سبب قويًا لتغييره — وتغييره قبل يومين من التحكيم مخاطرة بلا عائد.

```css
--fs-display:38px;  --fs-title:30px;    --fs-section:20px;
--fs-card:16px;     --fs-body:15px;     --fs-label:13px;
--fs-caption:12px;
```

### ⭐ الأرقام المالية
```css
font-variant-numeric: tabular-nums;
```
**لماذا؟** بدونها، عند تغيّر ٤٬٢٠٠ → ٤٬٠٠٠ **يتغيّر عرض الرقم** فترقص الصفحة.
مع `tabular-nums` كل رقم بنفس العرض — الانتقال نظيف.

مُطبَّق على: الرصيد · بطاقات المؤشرات · مبالغ العمليات · الفواتير · الخط الزمني.

## 7. الحركة — كل حركة تجيب على سؤال

```css
--dur-instant:90ms     /* ضغط زر → «سُجّلت نقرتك» */
--dur-fast:160ms       /* hover · خروج صفحة */
--dur-normal:240ms     /* دخول صفحة · تغيّر رقم */
--dur-emphasis:360ms   /* ظهور النتيجة */
--dur-entrance:450ms   /* دخول أول للوحة */
```

| الحركة | السؤال الذي تجيب عنه |
|---|---|
| انتقال الصفحة | «أين انتقلت؟» |
| تغيّر الرقم | «ماذا تغيّر؟» |
| إبراز البطاقة | «ما الذي تأثر؟» |
| حالة الزر | «هل سُجّل فعلي؟» |
| شارة الخطر | «هل تغيّر مستوى الخطر؟» |

**مرفوض:** confetti · نبض مستمر · parallax · دوران 3D · جسيمات · أصوات.
**السبب:** منتج مالي. الثقة تُبنى بالهدوء، لا بالاستعراض.

### 🛡 الحركة لا تُسقِط الوظيفة أبدًا
```js
/* ❌ ممنوع */
animationend → then save

/* ✅ المُطبَّق */
save → update state → animate result
```
لو فشلت طبقة الحركة بالكامل، **التنقل والحفظ والتحليل تعمل كما هي.**

## 8. حالات المكوّنات

كل عنصر تفاعلي: **Default · Hover · Pressed · Focus · Disabled · Loading**

```css
.btn:hover:not(:disabled){transform:translateY(-1px)}
.btn:active:not(:disabled){transform:scale(.98)}
.btn:disabled{opacity:.55;cursor:not-allowed}
.btn:focus-visible{box-shadow:var(--sh-focus)}
```

**بطاقات تفاعلية فقط ترتفع** (`.alt-card` · `.qa` · `.quick`).
البطاقات الساكنة لا تقفز — وإلا فقد الـhover معناه.

## 9. إمكانية الوصول

| البند | التنفيذ |
|---|---|
| تركيز مرئي | `:focus-visible` + `--sh-focus` على كل عنصر |
| لوحة المفاتيح | Tab · Escape · Enter للحفظ |
| حصر التركيز | Tab لا يخرج من النافذة |
| إرجاع التركيز | يعود للزر الذي فتح النافذة |
| قفل التمرير | + شبكة أمان كل 1.5s تمنع التعليق |
| أهداف اللمس | ≥44px على الجوال |
| اللون ليس وحده | لون + أيقونة + نص + إشارة |
| تقليل الحركة | `@media (prefers-reduced-motion: reduce)` |
