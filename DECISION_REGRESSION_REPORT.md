## 2. ⭐ السبب الجذري #1 — شاشة التحميل اختفت من المنصة كلها

### الانحدار

في **Startup Hardening (v2.2)** كتبتُ:

```js
function enterAppSafely(){
  const ld = document.getElementById('loading');
  ld.classList.add('hide');
  ld.style.display = 'none';        // 🔴 inline style — قسري
}
```

وضعتُ `style.display = 'none'` **مباشرة على العنصر** كاحتياط ضد Loading الأبدي.

لكن `showLoading()` كان:

```js
function showLoading(text){
  $('#loading').classList.remove('hide');   // ← يُزيل الـclass فقط
}
```

**الـinline style يتفوّق على أي CSS class.**

⇒ `showLoading()` صار **بلا أثر إطلاقًا**.
⇒ **شاشة التحميل لم تعد تظهر أبدًا — لا عند التحليل ولا عند التنقل.**

### الإثبات

عيّنتُ `display` كل 40ms بعد الضغط على «تحليل الأثر»:

```
none,none,none,none,none,none,none,none,none,none,none, ... (26 عينة)
```

**لم تظهر ولو لإطار واحد.**

### لماذا يبدو هذا كـ«الزر لا يعمل»؟

**قبل الانحدار (النسخة الأصلية):**
```
ضغط → شاشة تحميل مميزة (شعار قرار × الإنماء) → نتيجة
      ↑ ردّ فعل بصري فوري وواضح
```

**بعد الانحدار:**
```
ضغط → ...لا شيء... → النتيجة تظهر أسفل الصفحة (خارج الرؤية)
      ↑ صمت تام — يبدو أن الزر ميت
```

### الإصلاح

```js
function showLoading(text){
  const el = $('#loading');
  el.style.display = '';            // 🔴 امسح الـinline style أولًا
  el.classList.remove('hide');
}
```

**النتيجة:** شاشة التحميل عادت — **في التحليل وفي كل التنقلات**.

## 3. السبب الجذري #2 — النتيجة تظهر بلا رأس واضح

```
scrollY : 1307      ← الصفحة قفزت 1300px
top     : -19       ← رأس النتيجة فوق حافة الشاشة
height  : 2104      ← كتلة النتيجة طولها 2104px
```

`scrollIntoView({block:'nearest'})` على كتلة بهذا الطول كان يضع المستخدم
**في منتصفها** — فيرى جدارًا من البطاقات **بلا عنوان يقول إن هذه نتيجة**.

### الإصلاح
- رأس صريح: **«نتيجة التحليل» → «تحليل أثر القرار» → اسم القرار + المبلغ + شارة الخطر + التوصية**
- التمرير إلى **رأس النتيجة** (`block:'start'`)، لا إلى منتصف الكتلة
- قسم **«لماذا؟»** يشرح النتيجة بلغة طبيعية من الأرقام الفعلية
- صندوق **التوصية** منفصل وواضح

```js
doAnalysis();          // ✅ استدعاء مباشر ومتزامن
```

**لو تعطّل كل مؤقت في المتصفح:**
- التحليل ✓ تمّ

- شاشة التحميل ⚠️ قد تبقى — لكن **شبكة أمان (2.2s)** تُخفيها

**العكس مستحيل:** لا يمكن أن تكون هناك شاشة تحميل بلا نتيجة.
